# Welcome to Minimal Tray Tasker

Minimal Tray Tasker is a lightweight desktop application that helps you stay on track with your daily tasks. Built with Tauri and Svelte, it lives in your system tray and provides quick access to your task list without interrupting your workflow.

## Key Features

<CardGroup cols={2}>
  <Card title="System Tray Integration" icon="window-minimize">
    Access your tasks instantly from the system tray. The app stays out of your way until you need it.
  </Card>
  
  <Card title="Daily & One-Time Tasks" icon="calendar-check">
    Create tasks that reset daily or one-time tasks that stay until completed. Daily tasks automatically reset at midnight.
  </Card>
  
  <Card title="Hourly Reminders" icon="bell">
    Get gentle notifications every hour to remind you of pending tasks. Can be disabled in settings if you prefer.
  </Card>
  
  <Card title="Customizable Settings" icon="sliders">
    Control autostart, reminders, and performance with Ram Saver mode. Switch between light and dark themes.
  </Card>
</CardGroup>

## Perfect For

- Tracking daily habits like drinking water or taking breaks
- Managing simple to-do lists without heavyweight project management tools
- Quick task tracking that doesn't require opening a full application
- Staying reminded of important tasks throughout the day

## Tech Stack

Minimal Tray Tasker is built with:
- **Tauri 2.x** - Lightweight desktop framework
- **Svelte 5** - Modern reactive UI framework
- **SQLite** - Local database for task storage
- **TailwindCSS** - Utility-first styling
